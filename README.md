## Módulos utilizados

```
1- mkdir backend
2- yarn init -y
3- yarn add express
    acelerar a construcao da aplicação
4- yarn add nodemon -D
    para auxiliar na deteccao de alteracoes no arquivo
5- yarn add mongoose 
    facilitar trabalhar com dados no mongodb
6- yarn add cors
```
